# Sitebrief — Marketing UI Kit

A small, focused recreation of the marketing-site surface for Sitebrief.

## What's here

- `index.html` — sample landing page assembled from the components below
- `Hero.jsx` — left-anchored editorial hero with a single product specimen on the right
- `Nav.jsx` — top nav bar with logo, links, login, primary CTA
- `BriefSpecimen.jsx` — the static pitch-card used as the hero's right-side visual proof
- `FeatureGrid.jsx` — 3-up feature row with caption / title / body
- `LogoStrip.jsx` — "trusted by" muted logo wall
- `CTASection.jsx` — closing call-to-action band
- `Footer.jsx` — paper footer with rule pattern, mark, links

## Notes

This is a recreation of the *direction* described in `../../README.md`, not a recreation of an existing marketing site (none was provided). The goal is to make the visual rules concrete enough that anyone authoring a new page can copy the layout patterns wholesale.
