# Sitebrief — Dashboard UI Kit

The logged-in operator surface.

## What's here

- `index.html` — sample dashboard view (Site Fleet + Brief Inbox)
- `Sidebar.jsx` — fixed 240px navigation
- `Topbar.jsx` — global header with breadcrumb, search, user
- `SiteFleet.jsx` — grid of `SiteTile`
- `SiteTile.jsx` — per-site card with live crawl status
- `QueueMonitor.jsx` — three-queue worker widget
- `BriefInbox.jsx` — list of `PitchCard`
- `PitchCard.jsx` — the canonical decision-ready brief
- `FilterBar.jsx` — chip filters above the inbox

## Notes

UI was inferred from section 8 of the source project plan (Frontend-Anforderungen). No screenshots existed; this is a v0 to react against.
